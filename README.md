# Eye Tracker

Makes use of our novel eye localization method. Demo is prepared using HTML5 webRTC capture, face-api.js, opencvjs.

# Build

```
npm i express
npm i express-session
npm i express-password-protect
```

# Run
```
node index.js
```

Open https://localhost:8080/eyetracker
